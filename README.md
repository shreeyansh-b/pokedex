# Pokedex

Pokedex is a web app to find your favourite Pokemon. 

## Demo

[Pokedex](https://shreeyansh-b.github.io/pokedex/)


# Technologies Used
* React.js
* Redux.js
* Material-UI 
* Redux-thunk
* Axios
* React-Spring
* [Fast Average Color](https://www.npmjs.com/package/fast-average-color) 
* [Color](https://www.npmjs.com/package/color)
